### Color Combinations for Contrast

```
Color pairs that work together (WCAG compliant):

Pass/Fail:        #22C55E / #EF4444  (4.5:1 contrast on white)
Passed/Pending:   #22C55E / #F59E0B  (distinguishable)
Multi-series:     Use alternating from primary palette with sufficient spacing
```

  /* Chart primary colors */

## Animation Specifications

### Chart Transitions

```
Line Chart:
- Draw: 600ms ease-out, from left to right
- Point appear: 200ms fade + scale
- Value change: 300ms ease-in-out

Bar Chart:
- Grow: 400ms ease-out per bar
- Stagger: 50ms between bars
- Hover: 150ms scale(1.02)

Donut/Pie:
- Grow: 500ms ease-out, segments staggered 100ms
- Hover: 150ms scale(1.05) on segment
- Value change: 300ms transition

Progress:
- Value change: 300ms ease-in-out
- Circular: Smooth stroke-dashoffset transition
```

### Reduced Motion

```css
@media (prefers-reduced-motion: reduce) {
  .chart-animated {
    animation: none;
    transition: none;
  }

  /* Show final state immediately */
  .chart-line { stroke-dasharray: none; }
  .chart-bar { transform: none; }
}
```

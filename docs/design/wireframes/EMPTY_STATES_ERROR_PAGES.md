## Illustrations

### Illustration Style Guide

```
Style:
- Line art with minimal fills
- Primary color (indigo) accents
- Friendly, approachable aesthetic
- Consistent 2px stroke weight
- Rounded corners and soft shapes

Size Variations:
- Large: 128px for full-page empty states
- Medium: 96px for card empty states
- Small: 64px for compact empty states
- Icon: 48px for inline empty states

Color Usage:
- Outlines: neutral-400
- Accent: primary-500
- Fill (sparse): primary-100 or neutral-100
```

### Animation Hints

```
Illustrations can have subtle animations:

Floating:
- Slight up/down movement (4px range)
- Duration: 3s, ease-in-out, infinite
- Use for: Friendly empty states

Pulsing:
- Scale 1.0 to 1.05
- Duration: 2s, ease-in-out, infinite
- Use for: Loading states

Wiggle:
- Rotate -2deg to +2deg
- Duration: 0.5s, ease-in-out
- Use for: Error states (shake effect)

Fade-in:
- Opacity 0 to 1
- Duration: 300ms, ease-out
- Use for: All illustrations on load

@media (prefers-reduced-motion: reduce):
- Disable all animations
- Show static illustrations
```

| Error | Title | Description | CTA |
|-------|-------|-------------|-----|

### Tone

- **Empathetic**: Acknowledge the user's situation
- **Helpful**: Provide clear next steps
- **Concise**: Keep text short and scannable
- **Friendly**: Use conversational language
- **Never blame**: Don't make users feel at fault

## Accessibility

### Screen Reader Considerations

```html
<!-- Empty state with proper ARIA -->
<section aria-labelledby="empty-title" role="status">
  <img src="illustration.svg" alt="" aria-hidden="true" />
  <h2 id="empty-title">No packages yet</h2>
  <p>Create your first QA package to start testing your APIs.</p>
  <button>Create Package</button>
</section>

<!-- Error page -->
<main role="alert" aria-labelledby="error-title">
  <h1 id="error-title">404 - Page not found</h1>
  <p>The page you're looking for doesn't exist.</p>
  <nav aria-label="Error recovery options">
    <a href="/">Go to Dashboard</a>
  </nav>
</main>
```

### Focus Management

- Error pages: Focus on heading when page loads
- Empty states: Focus on CTA button if actionable
- Connection banners: Don't steal focus, use `aria-live`

### Color Contrast

- All text meets WCAG AA (4.5:1 for normal, 3:1 for large)
- Illustrations have sufficient contrast for meaning
- Status indicators don't rely on color alone

## Interaction Notes

### Navigation

- Active nav item has primary background
- Hover shows neutral-100 background
- Icons always visible, labels show on expanded state or hover

### Cards

- Hover elevates card (shadow-md)
- Click anywhere on card navigates to detail (except action buttons)
- Action buttons stop propagation

### Modals

- Click backdrop or press ESC to close
- Focus trapped inside modal
- First focusable element receives focus on open
- Submit with Enter when form is valid

### Loading States

- Full-page: Centered spinner with text
- Cards: Skeleton matching card layout
- Buttons: Spinner replaces icon, label stays
- Tables: Skeleton rows

### Error States

- Toast for transient errors (auto-dismiss)
- Inline for form validation
- Full-page for critical errors (with retry)

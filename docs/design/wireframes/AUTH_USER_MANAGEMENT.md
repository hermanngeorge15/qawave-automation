## Accessibility Notes

### Login Form
- Form has `role="form"` with accessible name
- All inputs have associated labels
- Error messages linked via `aria-describedby`
- Password visibility toggle has `aria-label`
- Loading states announced via `aria-busy`

### User Menu
- Menu button has `aria-haspopup="menu"`
- Menu has `role="menu"` with `aria-labelledby`
- Menu items have `role="menuitem"`
- Focus trapped in open menu
- Escape closes menu

### Session Cards
- Cards are in a list with `role="list"`
- Each card has `role="listitem"`
- Current session marked with `aria-current="true"`
- Actions are keyboard accessible

### Dark Mode Toggle
- Toggle has `role="switch"` with `aria-checked`
- Label describes current state
- Keyboard activatable with Space/Enter

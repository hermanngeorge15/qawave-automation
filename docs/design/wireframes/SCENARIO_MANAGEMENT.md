### JSON Editor Features

```
Syntax Highlighting:
  - Keys: blue
  - Strings: green
  - Numbers: orange
  - Booleans: purple
  - Null: gray
  - Brackets: default

Line Features:
  - Line numbers
  - Code folding (collapse objects/arrays)
  - Error highlighting (red underline)
  - Current line highlight

Keyboard Shortcuts:
  - Cmd/Ctrl + F: Find
  - Cmd/Ctrl + G: Go to line
  - Cmd/Ctrl + Shift + F: Format JSON
  - Cmd/Ctrl + Z: Undo
  - Cmd/Ctrl + Shift + Z: Redo
```

## Accessibility Notes

### Scenario Cards
- Cards are focusable with `tabindex="0"`
- Arrow keys navigate between cards
- Enter opens detail view
- Space toggles selection

### Step Editor
- Drag handles have `aria-label` describing action
- Reorder announced via `aria-live`
- Tab order follows visual order
- Focus trapped in modals

### JSON Editor
- Line numbers are decorative (`aria-hidden`)
- Errors announced via `aria-live`
- Keyboard navigation for all actions
- High contrast syntax highlighting

### Forms
- All inputs have associated labels
- Required fields marked with `aria-required`
- Error messages linked via `aria-describedby`
- Focus moves to first error on submit

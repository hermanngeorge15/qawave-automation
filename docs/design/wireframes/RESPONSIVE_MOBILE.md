## Breakpoints & Grid System

### Breakpoint Definitions

| Breakpoint | Name | Range | Columns | Gutter | Margin |
|------------|------|-------|---------|--------|--------|
| xs | Mobile (small) | 0 - 479px | 4 | 16px | 16px |
| sm | Mobile (large) | 480 - 639px | 4 | 16px | 16px |
| md | Tablet | 640 - 1023px | 8 | 24px | 24px |
| lg | Desktop | 1024 - 1279px | 12 | 24px | 32px |
| xl | Large Desktop | 1280px+ | 12 | 32px | 48px |

### CSS Custom Properties

```css
:root {
  /* Breakpoints */
  --breakpoint-xs: 0px;
  --breakpoint-sm: 480px;
  --breakpoint-md: 640px;
  --breakpoint-lg: 1024px;
  --breakpoint-xl: 1280px;

  /* Container widths */
  --container-xs: 100%;
  --container-sm: 100%;
  --container-md: 100%;
  --container-lg: 960px;
  --container-xl: 1200px;

  /* Grid */
  --grid-columns-mobile: 4;
  --grid-columns-tablet: 8;
  --grid-columns-desktop: 12;
}
```

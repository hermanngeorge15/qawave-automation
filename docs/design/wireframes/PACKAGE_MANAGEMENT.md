## Accessibility Notes

### Package Card
- Card is focusable with `tabindex="0"`
- Actions are keyboard accessible
- Status is announced via `aria-live`
- Metrics have proper labels for screen readers

### Filter Bar
- All dropdowns are keyboard navigable
- Active filters announced when changed
- Clear filter buttons have accessible labels

### Pagination
- Current page announced via `aria-current="page"`
- Navigation buttons have proper labels
- Page input allows direct number entry

### Bulk Actions
- Selection count announced
- Action bar traps focus when visible
- Clear button returns focus to list

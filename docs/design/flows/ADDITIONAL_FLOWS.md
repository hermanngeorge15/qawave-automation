### Key Interactions

| Step | Component | Behavior |
|------|-----------|----------|
| Tab Switch | Tabs | Instant switch, no page reload |
| Toggle | Toggle | Immediate save, optimistic update |
| Text Input | Input | Debounced save indicator |
| Save | Button | Loading state, success toast |
| Revoke Session | Button | Confirmation modal, then revoke |
| Create Token | Modal | Show token once, cannot view again |

### Key Interactions

| Step | Component | Behavior |
|------|-----------|----------|
| Select Item | Checkbox | Add to selection, show bulk bar |
| Shift+Click | Checkbox | Range select |
| Select All | Header Checkbox | Select current page |
| Clear | Button | Clear selection, hide bulk bar |
| Action | Button | Open action modal |
| Confirm Action | Modal | Execute action, show progress |

### Keyboard Shortcuts

| Shortcut | Action |

### Key Interactions

| Step | Component | Behavior |
|------|-----------|----------|
| Search Input | Input | Debounced (300ms), suggestions on focus |
| Suggestion Select | Dropdown Item | Navigate to item or apply search |
| Filter Toggle | Checkbox/Radio | Instant filter (no apply needed) |
| Clear Filter | Chip × | Remove single filter |
| Clear All | Button | Remove all filters |
| Sort | Dropdown/Header | Instant sort |
| Save Filter | Modal | Persist filter combination |

### URL Synchronization

Search and filter state is synced with URL for shareability:

```
/packages?search=payment&status=failed&tag=critical&sort=lastRun:desc
```

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| / or Ctrl+K | Focus search input |
| Escape | Clear search, close dropdowns |
| Arrow Down/Up | Navigate suggestions |
| Enter | Apply search/select suggestion |
| Ctrl+Shift+F | Open advanced filter panel |

## Accessibility Considerations

### Search
- Search input has `role="searchbox"` and `aria-label`
- Suggestions use `role="listbox"` with `aria-activedescendant`
- Clear button has `aria-label="Clear search"`

### Filters
- Filter panel uses `role="dialog"` when expanded
- Checkboxes properly labeled
- Active filters announced to screen readers

### Bulk Actions
- Selection count announced on change
- Bulk action bar uses `role="toolbar"`
- Actions have descriptive `aria-label`
